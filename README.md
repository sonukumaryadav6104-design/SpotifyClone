# 🎧 Spotify Clone – Frontend Music Player

> 🚀 A Spotify-inspired frontend application with interactive UI and music player controls.

---

## 📌 Overview

This project is a **frontend clone of Spotify**, designed to replicate the look and feel of a modern music streaming platform.
It focuses on **UI/UX design, responsiveness, and interactive music controls**.

---

## 🎯 Features

* 🎵 Music player UI
* ▶️ Play / Pause functionality
* ⏭️ Next / Previous controls
* 📂 Playlist interface
* 📱 Responsive design

---

## 🧠 Key Concepts

* DOM Manipulation
* Event Handling
* Responsive UI Design
* Component-based layout
